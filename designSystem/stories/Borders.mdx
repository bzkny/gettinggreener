#### Code Snippets
<p>There is a mixin available for borders as well as helper classes.</p>

Mixin:
```
@mixin border($width: 2px, $style: solid, $color:palette.$color-tertiary) {
    border: $width $style $color;
}
```

Classes:
```
.border-primary-brand {
  @include border();
}
```
